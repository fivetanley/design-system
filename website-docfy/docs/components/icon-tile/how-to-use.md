# IconTile - How to use

Invocation of the component would look something like this:

```handlebars
<Hds::IconTile @logo="boundary" @size="medium" iconSecondary="plus" />
```

```handlebars
<Hds::IconTile @icon="dashboard" @size="small" @color="vault" />
```
